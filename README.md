본 프로젝트는 Handong Global University(한동대학교)와 KAPPD의 산학협력을 통해 진행되는 프로젝트입니다.

<br>

## 📖 프로젝트 소개

### 제안 배경

희망이음 등 기존 시스템은 **일반 종사자 중심 설계**로, 근로장애인의 개별 특성을 반영하지 못합니다.

> 근로장애인은 완전한 종사자가 아닌 근로자이자 지원 대상입니다.  
> 무급 훈련일이 있거나 근로 패턴이 일정하지 않은 개별 상황을 기존 시스템은 다루기 어렵습니다.

또한 공단에서 제공하는 엑셀 파일을 시설 내부 양식에 **매달 수기로 옮기는 반복 작업**이 행정 낭비로 이어지고 있으며, 기성 프로그램으로는 구현하기 어려운 시설 맞춤형 로직의 부재로 인해 구조적 행정 사각지대가 발생하고 있습니다.

**기존 업무 흐름**
```
공단 엑셀 수령 → [수기 데이터 옮기기 ⚠️] → 시설 양식 완성 → 급여 계산 · 명세서 발급
```

### 제안 내용

시설 맞춤형 데이터 매칭을 통해 행정 사각지대를 없애고, 단순 반복 업무를 디지털로 전환(DX)합니다.

| 기능 | 설명 |
|------|------|
| **데이터 브릿지 자동 매칭** | 공단 엑셀 파일 업로드 → 시설 급여 대장 자동 매칭, 수기 작업 완전 제거 |
| **맞춤형 급여 자동 차감** | 무급 일수 입력 시 급여 자동 차감, 근로장애인 개별 근로 패턴 반영 |
| **서류 원클릭 발급** | 개인별 급여 명세서 · 재직증명서를 클릭 한 번으로 발급 |

**개선된 업무 흐름**
```
공단 엑셀 업로드 → [자동 매칭 · 급여 계산 ✅] → 명세서 · 재직증명서 발급
```
